图形界面方式

Ubuntu 18.04 在系统设置里面是找不到软件源设置界面按钮的(找了好久都没找到T-T)，
所以需要一些指令帮助启动图形界面以便设置软件源。

1、输入命令

2、会弹出 Software Updater 提示框，等待一会儿后，
点击左下角 Settings… 按钮，久违的软件源设置就会出来啦。

3、选择

  1. 选择 Ubuntu Software
  2. 选择 Download from:
  3. 选择 Other…
  4. 选择 Select Best Server
