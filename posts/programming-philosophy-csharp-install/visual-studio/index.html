工欲善其事必先利其器，本章介绍安装Visual Studio这个号称宇宙最强IDE(Integrated Development Environment[集成开发环境])。

安装 Visual Studio

Visual Studio官网下载地址

点击 免费下载 下载完毕后得到一个 vs_community.exe 文件，运行 vs_community.exe

安装过程如下：

直接点 继续

等待稍等片刻，就会弹出我们需要我们选择安装的内容…

如图请首选上.NET桌面开发

毕竟.NET Core才是未来，所以强烈建议勾选上 .Net Core跨平台开发 。

这个时候安装程序就会下载你刚刚勾选需要安装的工具。而截图中左边会出现 Visual Studio Enterprise 2017 是因为本人安装了vs的企业版本。

先跳过，选择 以后再说 。要注意的是，30天后需要你用微软的账号登陆，到时候注册一个就好了。

默认是 蓝色 ，建议选择 深色 ,这样对眼睛会好一点，特别是晚上敲代码的时候。如果想要切换主题，请在Visual Studio的菜单栏点击 工具 → 选项 在弹出来的子窗口中，环境 下的 常规 右边颜色主题，重新选择就好了。

结束语

本章介绍Visual Studio(以后一般情况下都会简称为VS)的安装过程。接下来打铁成热，为了检查我们安装的工具是否成功，我们赶紧试试著名的”Hello,World!”例子吧！

Hello, World是指在计算机屏幕显示”Hello,World!”（你好，世界！）字符串的计算机程序。相关的程序通常都是每种计算机编程语言最基本、最简单的程序，也会用作示范一个编程语言如何运作。同时它亦可以用来确认一个编程语言的编译器、程序开发环境及运行环境是否已经安装妥当。因为写法简单可见，这也是很多初学者首次接触编程语言时会撰写的程序。
